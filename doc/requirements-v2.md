# Video Mosaic Tool - 要件定義書 v2

**変更**: 2026-03-25  
**変更内容**: 動画プレビュー → 静止画フレーム + UI シンプル化

---

## 1. 機能要件（更新）

### F1: 動画ファイル選択
- ユーザーがローカルファイルから動画を選択
- ファイル読み込み → 動画メタデータ認識

### F2: 1フレーム目を静止画として切り出す
- **新機能**: 動画の **1フレーム目を Canvas に描画**
- `canvas.drawImage(video, 0, 0)` で静止画を取得
- PNG または JPEG として画像表示
- **メリット**: iPhone での領域選択が簡単になる

### F3: 領域選択（静止画上）
- 静止画（`<img>`）上でドラッグして四角形を描く
- マウス/タッチ対応
- ドラッグ中: 青い枠、離時: 黄色い枠
- **1つの領域のみサポート**

### F4: ぼかし強度スライダー
- 0～100% で調整
- デフォルト: 50%

### F5: モザイク処理実行
- FFmpeg で動画全体をぼかし
- プログレスバー表示

### F6: 動画ダウンロード
- 自動ダウンロード（iOS Safari 対応）

## 2. 技術仕様（変更）

### 2.1 1フレーム目の抽出

```typescript
const extractFirstFrame = async (videoFile: File): Promise<Blob> => {
  return new Promise((resolve, reject) => {
    const video = document.createElement('video')
    const canvas = document.createElement('canvas')
    
    video.onloadedmetadata = () => {
      canvas.width = video.videoWidth
      canvas.height = video.videoHeight
      
      const ctx = canvas.getContext('2d')
      ctx?.drawImage(video, 0, 0)
      
      canvas.toBlob((blob) => {
        if (blob) resolve(blob)
        else reject(new Error('Canvas to Blob failed'))
      }, 'image/png')
    }
    
    video.src = URL.createObjectURL(videoFile)
  })
}
```
